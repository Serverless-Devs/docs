# Gui指令帮助文档

通过`s gui`指令，您可以通过该指令唤起gui可视化操作页面（即Serverless Devs App Store）

- [基础能力](#基础能力)
- [帮助信息](#帮助信息)

## 基础能力

![](https://images.serverlessfans.com/s-gui/docs/app-store-index.jpg)

如果您需要对Gui进行升级，可使用：

```
s gui --update
```
